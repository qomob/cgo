## Layer 1: Growth Strategy Core

### Growth System Design
- User growth funnel (AARRR)
- Growth lever identification (where AI amplifies)
- Growth flywheel design (data -> decision -> automation)

### Business Model Reconstruction
- AI-driven SaaS / service / platform restructuring
- Cost structure optimization (AI replacing manual labor)
- Revenue model design (subscription / usage / outcome-based)

### Strategic Judgment
- AI opportunity identification
- ROI evaluation model
- Tech vs. business priority tradeoff

## Layer 2: AI System Capability

### AI Workflow Design
- Prompt -> Workflow -> Agent -> System
- RAG system design (knowledge augmentation)
- Multi-Agent collaboration (Planner / Executor / Evaluator)

Core principle: transform "AI tools" into "business systems".

### AI Productization
- AI feature embedding into products (not demos)
- AI feature -> business feature conversion
- Copilot / Autopilot design

### Data & Model Understanding
- LLM capability boundary judgment
- Hallucination risk control
- Embedding / vector search / fine-tuning basics

## Layer 3: Growth Execution

### AI-Driven Marketing
- AI content generation system (SEO / Ads / Social)
- Automated ad optimization
- User segmentation and personalized outreach

### Conversion Optimization
- Landing page A/B testing
- Conversion funnel optimization
- AI behavior prediction (churn / conversion)

### Sales System AI-ification
- AI SDR (automated sales assistant)
- Lead scoring system
- Automated follow-up and CRM intelligence

## Layer 4: Data Flywheel

### Growth Data System
- North Star metric design
- Real-time growth dashboard
- User behavior event modeling

### AI Feedback Learning
- Human feedback loop
- Reinforcement signals (conversion / retention)
- A/B test automated decision system

### Attribution Analysis
- Multi-touch attribution
- AI strategy effect attribution
- Growth factor decomposition

## Layer 5: Organization & Orchestration

### Cross-Department AI Collaboration
- Product / Tech / Marketing / Sales unified AI language
- Break department data silos
- AI project implementation mechanism

### AI Governance
- AI risk control (hallucination / data leakage)
- AI compliance strategy
- Model usage standards

### Organization Upgrade
- Human-machine collaboration structure design
- AI-displaced role restructuring
- AI-driven org design

## Layer 6: Advanced / Next Level

### Autonomous Growth System
- Self-optimizing funnel
- Autonomous marketing system
- AI-driven decision engine

### Multi-Agent Growth System
- AI team replacing growth team
- Planner / Executor / Analyst agents
- Agent orchestration system

### Real-time Growth OS
- Company growth operating system (Growth OS)
- Real-time strategy adjustment system
- Continuous optimization loop

## Role Differentiation

| Role | What They Do |
|------|-------------|
| AI Engineer | Build models / systems |
| AI Product Manager | Build AI features |
| AI CGO | Use AI to make the company more money |

Engineers build tools. CGO builds results.
